\section{Ablation Studies}
\label{sec:ablations}

Table~\ref{tab:ablation} reports single-axis ablations of CSM-SAM on
HNTS-MRG 2024 Task 2. Each row removes or replaces one component:
cross-session memory is disabled to recover a MedSAM2-style
within-session propagator~\cite{ma2024medsam2}; the change head is
removed to test whether the XOR auxiliary signal is load-bearing; the
continuous weeks-elapsed embedding is removed to test temporal
calibration; and, as a non-attention control, pre/mid are fed as a
dual-modality input to the SAM2 encoder instead of via memory.

\subsection{Variational Change Latent}
\label{sec:ablations:latent}

Table~\ref{tab:ablation_latent} isolates each component of the variational
change latent.  All rows use the same base model (cross-session attention +
change head + continuous time); only the latent module changes.

\centering

\noindent The critical row is the Spearman $\rho$ column of the full model:
a value above $0.5$ with $p < 0.01$ on held-out patients would confirm that
$\boldsymbol{\mu}_q$ encodes treatment response phenotype without any
explicit phenotype labels in the training signal.
