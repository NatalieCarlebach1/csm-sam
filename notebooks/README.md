# Notebooks

Exploratory notebooks for CSM-SAM.

| Notebook | Description |
|----------|-------------|
| `01_data_exploration.ipynb` | Visualize HNTS-MRG data, check pre/mid-RT pairs |
| `02_architecture_walkthrough.ipynb` | Step through CrossSessionMemoryAttention |
| `03_training_curves.ipynb` | Plot training metrics from training_history.json |
| `04_ablation_analysis.ipynb` | Compare ablations: no cross-session, no change map |
| `05_error_analysis.ipynb` | Analyze worst-performing patients |

Run notebooks after downloading and preprocessing the data.
